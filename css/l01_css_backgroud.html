img요소는 inline-block속성의 독립적인 노드 개체다.

안녕하세요

background-image : 해당 요소의 스타일 속성, 이름부터 배경이미지
(background-repeat: repeat(default값)

안녕하세요

background-size : 요소와 관계없이 이미지의 가로 세로 크기 설정
(이미지가 여러개면 ,로 구분하여 설정을 따로 줄 수 있다.)

background-contain : 이미지 비율을 유지하며 노드의 크기만큼 크기 조절
(사진이 잘리지 X 때까지)

background-cover : 이미지 비율을 유지하며 노드 전체를 덮을 만큼 크기 조절
(사진이 자릴 수 O)

background-clip : content-box;
background-color영역을 content영역으로 제한

background-clip: padding-box;
background-color영역을 패딩까지
(border는 제외)

background-clip: border-box;
background-color영역을 border까지
(background의 default : content + padding + border)

background-origin: content-box;
왼쪽 상단을 기준으로 border, padding, content중 어느 라인부터 시작할지 선택

background-position : left, right, center, px, % / top, bottom, center, px, %

background-attachment: fixed = 브라우저 화면을 기준으로 위치가 고정
(결과적으로 이미지가 스크롤을 따라 움직인다.)

빨간블럭

background: bg-color bg-image position/bg-size bg-repeat bg-origin bg-clip bg-attachment initial|inherit;
