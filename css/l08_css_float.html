css float로 블럭 정렬하기

  • float는 text가 여러 줄 있을 때 그 왼쪽 or 오른쪽에 블럭을 삽입하기 위해 쓰임
  • 마치 position: absolute 처럼 부유객체 비슷하지만 text와 겹치지 않고 공간을 차지함.
  • text가 여러줄 있을 때는 display: inline-block을 사용하는 것은 적절치 X
  • inline-block은 말그대로 박스를 글자취급하여 박스의 좌우로 text가 달랑 한줄만 나란히 붙음.

float가 가진 문제

  • float블럭은 그것을 담은 container(부모노드)가 컨텐츠로 인식하지 못한다.
  • 뿐만 아니라 그 형제노드 들 역시 컨텐츠로 인식하지 못한다
  • float블럭 끼리는 서로를 인식하지만 위치관계를 조절하기 쉽지X
