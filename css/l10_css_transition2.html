transition이론 & 타이밍 함수

  • transition-property: 애니메이션이 가능한 css 속성(default: all)
  • transition-duration: 트렌지션이 일어나는 시간, s(초) or ms(밀리초)로 지정
  • transition-delay: 트렌지션이 시작되기 전 대기하는 시간
  • transition-timing-function: transition효과와 관련하여 속도를 함수로 지정
  • https://cubic-bezier.com/ : cubic-bezier(), ease, linear, ease-in, ease-out, ease-in-out

ProgressBar feat.dataset

  • dataset : 개발자가 요소에 직접 지정한 속성
  • 속성지정: 태그에 직접줄 때 data-속성이름="속성값"
  • 불러오기: element.dataset.속성이름 or element.dataset['속성이름']
  • 노드를 제어하는 방법으로 value가 쓰인 것 처럼, dataset을 유용하게 쓸 수 O
  • 예) 속성이름:key-goo -----> 불러오기: keyGoo
