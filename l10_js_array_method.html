자바스크립트 배열의 유용한 메소드

const al=["b","c","a","z","f","e","h"];

  • al.toString(): 배열의 값을 쉽표로 구분된 문자열로 출력한다.
  • al.join(/+/) : 배열의 값을 join안에 문자열로 결합된 문자열을 출력한다.????????????????????
  • al.pop() : 배열의 마지막 값을 반환하고 제거한다.
  • al.shift() : 배열의 첫번째 값을 반환하고 제거한다.(모든 인덱스가 한칸 당겨진다.)
  • al.push() : 배열의 마지막에 새 값을 추가한다.
  • al.unshift() : 배열의 첫번째에 새 값을 추가한다.
  • al[maxIndex+1]=LastVal : 마지막 배열에 새 값을 추가하는 또다른 방법
  • al.concat([1,3,2]) : 배열의 끝에 새로운 배열을 추가한 배열을 반환한다.(이때 al은 바뀌지X)
  • al.slice(start_index, end_index) : 시작점부터 끝점까지의 배열값을 반환한다.(제거가 X)

  • al.splice(delete_start index ,delet_length,insert_item,...) : 배열의 특정 element를 삭제하기

  • al.sort() : 배열의 요소를 순서대로 정렬한다.
  • al.reverse() : 배열의 요소를 역순으로 정렬한다.
