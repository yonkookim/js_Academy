할당 연산자와 증가 및 감소 연산자

operator 이름 설명 예시
a+=b 추가할당 a=a+b : 자신에게 자신+b의 값을 대입한다. x+=10 : x=x+10
a-=b 빼기할당 a=a-b x-=10 : x=x-10
a*=b 곱셈할당 a=a*b x*=10 : x=x*10
a/=b 분할할당 a=a/b x/=10 : x=x/10
++a; , --a; 증가 및 감소 연산자 a=a+1 , a=a-1 숫자 앞에 증가 및 감소 연산자(++, --)가 오면
연산을 먼저하고 다른실행을 옮긴다.		 function1(++i); : i를 증가시키고 function1을 실행
a++; , a--; 증가 및 감소 연산자 a=a+1 , a=a-1 숫자 뒤에 증가 및 감소 연산자(++, --)가 오면
다른실행을 먼저한 뒤 연산을 한다.		 function1(i++); : function1을 실행시키고 i를 1 증가



비교 연산자

참과 거짓을 테스트하는 비교 연산자

operator 이름 설명 예시
== 동등 a와 b의 값이 같음(을 비교) 10==6+4
!= 비동등 a와 b의 값이 같지 않음 10!=6+4
a===b 엄격한 동등 a와 b가 완전히 같음(value, type모두 고려) 10===new Number(6+4) :
10===new Number(6)+4 :
10==new Number(6+4) :
a!==b 엄격한 비동등 a와 b가 완전히 같지는 않음(value, type모두 고려) 10!==new Number(6+4) :
10!==new Number(6)+4 :
10!==new Number(6+4) :
a<b 미만(작다) a가 b보다 작다 5 < 3 : false
a>b 초과(크다) a가 b보다 크다 5 > 3 : true
a<=b 이하(작거나 같다)) a가 b보다 작거나 같다 5 <= 3 : false
a>=b 이상(크거나 같다) a가 b보다 크거나 같다 5 >= 3 : true
