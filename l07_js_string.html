문자들의 집합 문자열(String)

String 객체는 문자들의 집합인 문자열을 조작하는 함수를 제공한다.

문자열 생성 방법

  • let x= "abc"; (기본데이터 타입이다)
  • let x= 'abc'; (자바스크립트는 캐릭터 타입(Char)이 X)
  • let x= `abc`; (literal template 적용 문자열)

문자(char) 접근하기

  • "string"[0] :
  • "string".charAt(2) :

기본 데이터 타입인 문자열과 인스턴트 객체인 문자열

  • let stringVal="안녕하세요"; (기본 데이터 타입의 문자열)
  • let stringVal=new String("안녕하세요"); (new 연산자로 생성한 인스턴트 객체)
  • typeof stringVal :
  • typeof objectVal :
  • stringVal==objectVal : (value는 같지)
  • stringVal===objectVal : (type은 다르지))
  • 기본데이터 타입인 문자열에는 함수가 존재하지 않는다. 하지만 사용할 수 있는 이유는 사용할 때 인스턴트 객체 생성과 비슷한 일이 발생하기 때문이다.

템플릿 리터럴${표현식}

문자열을 ``백틱으로선언하면 템플릿 리터럴 ${} 표현식을 사용한다는 선언이다. 이때 자바스크립트 문법을 표현식에 사용할 수 있다.
