자바스크립트의 논리 연산자

&&(and), ||(or), !(not)

||(or) : 합집합(덧셈), 하나만 진실이어도 진실

&&(and) : 교집합(곱셈), 하나만 거짓이어도 거짓

논리연산의 순서(우선순위)

&&(곱셈) ||(덧셈) 수학연산과 동일하게 곱셈부터 한다 생각하면 돼! 그리고.. 그냥 논리적으로 당연

if 조건문

if(condition){} : 조건이 참이면 블럭을 실행한다.

if(condition){}else{} : 조건이 거짓이면 else블럭을 실행한다.

if(condition1){}else if(condition2){}else{} : 1번 조건 만족시 1번 블럭 실행 그렇지 않으면 2번 조건 실행, 모두 만족하지 않으면 else실행
