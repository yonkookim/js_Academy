Switch 조건문

작성법
            let 변수1=대입값;
            switch(변수1){ 
                case a:   
                --실행--
                break;

                case b: 
                --실행--
                break;

                case c: 
                --실행--
                break;

                default:
                --실행--
            }

스위치 케이스는 엄격한 비교(===)을 사용하고 위에서 부터 일치하는 항목을 찾으면
스위치는 종료된다.(엄격한 동등비교 외의 조건은 X)

if에서 else if문과 동일하게 작동하지만 조건문의 수행속도가 굉장히 빠르다.

시각적으로 조건을 파악하기 좋다.

삼향연산자

활용법1. (condition)? true : false <=> if(condition){true}else{false}

활용법2. let variable = (condition)? trueObject : falseObject;

활용법3.
            (condition)? true 
            : (condition2)? true 
            : (condition3)? true 
            : false(else)

조건파악이 편하고 코드가 간결해 프로그래밍 속도를 올려준다.
