typeof 는 기본데이터 타입 외에 대부분을 object로 검사한다. 이때 new Number()은 '수'이지만 기본데이터 타입이 아니다.
object는모든 기본데이터타입의 부모이다 : 모든 class??는 자동으로 object 확장자를 가지며. object는 인스턴스 객체의 생성을 돕는다?
ex) 100==new Number(100) : 값(value)을 비교
ex) 100===new Number(100) : 값과 type을 비교
