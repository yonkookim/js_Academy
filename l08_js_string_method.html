자바스크립트 문자열 string의 메소드

문자열은 유저에게 전달하는 데이터로 웹상에서 가장 많이 사용된다. 때문에 문자열을 다루는 함수가 많이 존재한다.

method 설명 예시
toUpperCase(), toLowerCase() 문자열의 대문자 or 소문자로 반환 "A b C d E".toUpperCase() :
"A b C d E".toLowerCase() :
trim() 문자열의 앞뒤 공백을 제거한다. " 최경민 ".trim().length :
charAt(), charCodeAt() "ABCDE".charCodeAt(0) :
"ABCDE".charAt(0) :
"ABCDE"[0] :
sclice(start, end) 일부 문자를 추출할 때 사용한다. "스파게티 피자 된장찌개 샌드위치".slice(8, 12) :
"스파게티 피자 된장찌개 샌드위치".slice(-9,-5) :
"스파게티 피자 된장찌개 샌드위치".slice(5) :
"스파게티 피자 된장찌개 샌드위치".slice(-5) :
"스파게티 피자 된장찌개 샌드위치".slice(13, 17) :
"스파게티 피자 된장찌개 샌드위치".slice(-4,) :
"스파게티 피자 된장찌개 샌드위치".slice(-4,) :
substring(start,end) slice와 동일하지만 양수 인텍스만 사용가능 "스파게티 피자 된장찌개 샌드위치".substring(5,7) :
substr(start,length 일부 문자를 추출한다. 이때 시작점부터 길이까지 추출한다. "스파게티 피자 된장찌개 샌드위치".substr(8,2) :
"스파게티 피자 된장찌개 샌드위치".substr(-9,2) :
"스파게티 피자 된장찌개 샌드위치".substr(-4,4) :
"스파게티 피자 된장찌개 샌드위치".substr(food.length-4, 4) :
startWith() 특정 문자열로 시작하는지 여부(boolean) food.startsWith("스파"):
food.startsWith("스파",0):
food.startsWith("된장",8):
endsWith(String, position=String.length) 특정 문자열로 끝나는지 여부 food.endsWith("위치") : food.endsWith("된장",10) :
includes(String, start position) 어떤 위치부터 특정 문자열이 있는지 여부(boolean) food.includes("찌개") :
food.includes("찌개" 12) :
indexOf(string, start position) 특정 위치부터 특정 문자열이 있는지 검색후 찾으면 바로 stop 문자의 시작점index를 반환, 없으면 -1을 반환 food.indexOf("피자") :
food.indexOf("피자") :
food.indexOf("스파게티") :
lastIndexOf(string, end position) 특정 위치부터 거꾸로 특정 문자열이 있는지 검색후 찾으면 바로 stop 문자의 시작점index를 반환, 없으면 -1을 반환 "Brave new world".lastIndexOf("w") :
"Brave new world".lastIndexOf("r") :
"Brave new world".indexOf("r") :
"Brave new world".lastIndexOf("new") :
split(String, limit length) 특정 문자열을 배열로 만들 수 있다.(길이를 정할 수 있다.) document.write(typeof food) :
document.write(typeof food.split(" ")) :
food.split(" ",3) :
food.split(",") :
replace() 특정 문자열이나 정규식 패턴으로 검색후(??) 다른 문자열로 변환한 값을 반환(바꾸는 것이X) food.replace("된장찌개","스테이크") :
food.replace("",",") :
food.replace("된장찌개","$& 제외 스테이크 추가") :
replaceAll() 특정 문자열이나 정규식 패턴으로 검색후(??) 모든 문자열을 다른 문자열로 변환한 값을 반환(바꾸는 것이X) food.replaceAll("",",") :
